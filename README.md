# TRION ARENA

『ワールドトリガー』のトリガー戦闘を題材にした、見下ろし型の非公式ブラウザゲームです。一般隊員同士の個人戦・チーム戦、フラッグを守る防衛戦、防衛戦ユニットを操作できるエキストラモードを遊べます。

> **対応バージョン:** 67  
> 本作は非公式ファンゲームです。公式画像、公式音源、公式ロゴ、公式ゲームデータは使用していません。

## ドキュメント

- **このREADME:** 起動、設定、操作、ルール、全体仕様
- **[攻略本](攻略本.md):** 装備例、戦術、マップ攻略、防衛戦ボス対策、エキストラ形態の技表

## 1. すぐに遊ぶ

### ローカル起動

1. リポジトリまたは配布ZIPを展開します。
2. `index.html` をChrome、Edge、Firefoxなどのブラウザで開きます。
3. タイトル画面で **「出撃準備へ」** を押します。
4. モード、マップ、能力、トリガーを選び、**「ランク戦を開始」** を押します。

ビルド、パッケージのインストール、サーバー起動は不要です。オンライン機能を使わない場合は、Supabase設定も不要です。

### GitHub Pages

静的ファイルだけで動作するため、リポジトリのGitHub Pagesを有効にすれば公開できます。ブランチは通常 `main`、公開フォルダはリポジトリ直下を指定します。

## 2. ゲームモード

### 個人戦

全員が敵です。撃破、アシスト、トリオン粒子の回収でポイントを獲得し、試合終了時の個人スコアを競います。

- CPU参加人数: 3～19人
- 試合時間: 2分、3分、5分、無制限
- 戦闘員、観戦で参加可能
- エキストラでは任意の参加者を防衛戦ユニットへ変更可能

### チーム戦

2～4チームでスコアを競います。各チームは1～4人の戦闘員とオペレーターで構成されます。

- 味方への攻撃は無効
- 個人の得点がチームスコアへ加算
- `O` でオペレーター指示パネルを開ける
- 戦闘員、オペレーター、観戦で参加可能

### 防衛戦

味方フラッグを守りながら、ラウンドごとに出現する侵攻ユニットを撃退します。フラッグの耐久が0になると敗北です。

防衛戦には2種類のシナリオがあります。

| シナリオ | 通常ラウンド | 5ラウンドごとのボス |
|---|---|---|
| ブラックトリガー | モールモッド、イルガー、ラービット | 風刃、印、アレクトール、ボルボロス、オルガノン |
| 百鬼夜行 | 骸骨アタッカー、骸骨シューター、骸骨スナイパー | 山狗、夜鴉、白狐、猫又、大蛇 |

防衛戦の敵は通常の隊員より高耐久です。敵は射線と技ごとの最大射程を考慮し、壁越しには攻撃しません。経路が塞がれている場合は迂回し、長時間進めない場合のみ破壊可能な壁を攻撃します。

### エキストラ

通常の個人戦・チーム戦・防衛戦を土台にし、プレイヤーやCPUを防衛戦ユニットへ変更するモードです。

設定できる項目:

- 基礎ルール: 個人戦、チーム戦、防衛戦
- 自分の形態
- CPUごとの形態
- 防衛戦で毎ラウンド追加する固定侵入ユニット

選択可能な形態:

- 隊員
- モールモッド、イルガー、ラービット
- 風刃、印、アレクトール、ボルボロス、オルガノン
- 骸骨アタッカー、骸骨シューター、骸骨スナイパー
- 山狗、夜鴉、白狐、猫又、大蛇

特殊形態も隊員と同じく、移動、照準、MAIN/SUB選択、左右クリックで操作します。各形態の全アクションは[攻略本](攻略本.md#10-エキストラ形態攻略)を参照してください。

## 3. 参加方法

| 参加方法 | 内容 |
|---|---|
| 戦闘員 | 自分でキャラクターを操作します。 |
| オペレーター | チーム戦・防衛戦で作戦指示と支援を行います。 |
| 観戦 | CPU戦を観戦します。`Q` / `E`で対象を変更できます。 |

試合中に `V` を押すと観戦状態を切り替えられます。自主ベイルアウト後も観戦できます。

## 4. PC操作

キー割り当ては出撃設定画面から変更できます。以下は標準設定です。

| 操作 | 標準キー |
|---|---|
| 移動 | `W` `A` `S` `D` |
| 照準 | マウス |
| MAIN 1～4を選択 | `1`～`4` |
| SUB 1～4を選択 | `5`～`8` |
| 選択中のMAINを発動 | 左クリック |
| 選択中のSUBを発動 | 右クリック |
| 合成弾・マンティス・形態固有操作 | `C` |
| 変形・分割・設置切替・盾・手動リロード | `Shift` |
| スコープ | `R` |
| スイッチボックス種類切替 | `Z` |
| 自主ベイルアウト | `B` |
| フラッグ修復 | `F` |
| 観戦切替 | `V` |
| 観戦対象を前後変更 | `Q` / `E` |
| オペレーター指示 | `O` |
| 戦闘ログ | `L` |
| 操作ガイド | `G` |
| 一時停止・メニュー | `P` / `Esc` |

### 長押しと組み合わせ

- シールド、レイガスト盾、印の盾は押している間だけ展開します。
- アサルトライフルとガトリングガンは長押しで連射します。
- 銃手・狙撃手は `Shift` を押しながら発動すると手動リロードします。
- 地下通路では、ホームドア、水門、ブレーカーの近くで特殊操作キーを押すと切り替えます。

### エキストラ形態の `C`

| 形態 | `C` の動作 |
|---|---|
| 印 | BOOSTを×1→×2→×3の順に切り替え |
| ボルボロス | 本物1個とデコイ5個のコアを再展開 |
| 風刃、アレクトール、オルガノン、山狗、夜鴉、白狐、猫又 | いなしを発動 |
| 大蛇 | 終焉の炎を発動 |

## 6. 能力配分

通常は合計18ポイントを、3種類の能力へ配分します。各能力は2～10です。

| 能力 | 主な効果 |
|---|---|
| トリオン総量 | 最大トリオン、回復、射程、弾出力 |
| 技術力 | 命中精度、誘導、合成速度、クールダウン、クリティカル |
| 戦闘力 | 移動速度、近接威力、耐久、武器の取り回し |

### ビギナーズスキル

スキルを選ぶと能力予算が12ポイントになります。

| スキル | 効果 |
|---|---|
| オートガード | シールドまたはレイガスト装備時、被弾直前に自動防御 |
| エイム補正 | 銃手・狙撃手の照準を近い敵へ弱く補正 |
| 倹約家 | すべてのトリオン消費を約18%削減 |

## 7. トリガーセット

右手MAIN 4枠、左手SUB 4枠の合計8枠へ装備します。

### 攻撃手用

| トリガー | 特徴 |
|---|---|
| 弧月 | 標準ブレード。`Shift`発動でいなし攻撃、斬撃直後にジャスト判定。 |
| 旋空 | 同じ側の弧月と組み合わせる長射程斬撃。 |
| スコーピオン | 軽量ブレード。`Shift`で普通・長・短を切り替え。両手装備時は`C`でマンティス。 |
| レイガスト | 高耐久ブレード。`Shift`長押しで盾。 |
| スラスター | 同じ側のレイガストと組み合わせる高速突進。 |

### 射手用

アステロイド、メテオラ、バイパー、ハウンドがあります。

1. 発動してキューブを展開します。
2. 展開中に `Shift`＋発動で最大5段階へ分割します。
3. 再発動して射撃します。
4. 反対側にも射手弾を展開し、`C`を押すと合成弾を作れます。

### 銃手用

アサルトライフル、ハンドガン、ショットガン、グレネードガン、ガトリングガンへ、アステロイド、メテオラ、バイパー、ハウンドを設定できます。

- 銃ごとに弾倉とリロード時間があります。
- 適正距離外で命中するとダメージが低下します。
- `Shift`＋発動で手動リロードできます。
- `R`でスコープを使用できます。

### 狙撃手用

| トリガー | 特徴 |
|---|---|
| イーグレット | 射程重視の万能型 |
| ライトニング | 低威力・高弾速・多弾数 |
| アイビス | 高威力・低弾速・少弾数 |

### 射撃オプション

| トリガー | 特徴 |
|---|---|
| 鉛弾 | 次に反対側から撃つ弾を、シールドを無視する拘束弾へ変更 |
| スタアメーカー | 次に反対側から撃つ弾へマーカーを付与 |

### 防御・隠密

| トリガー | 特徴 |
|---|---|
| シールド | 正面防御。左右同時展開で全周防御。 |
| エスクード | 照準地点へ破壊可能な硬い壁を生成。 |
| バッグワーム | レーダーと通常の敵視から外れる低消費隠密。 |
| バッグワームタグ | 片側4枠を占有する極低消費隠密。 |
| カメレオン | 姿を隠して敵視を大きく切る。使用中は消費が大きい。 |

### 機動・工作

| トリガー | 特徴 |
|---|---|
| スパイダー | 通常ワイヤーと、接触対象を跳ね飛ばすばねワイヤー。 |
| テレポーター | 照準方向へ瞬間移動。 |
| グラスホッパー | 照準方向へ急加速。接触した他の物体にも作用。 |
| ダミービーコン | 偽のトリオン反応でCPUとレーダーを撹乱。 |
| スイッチボックス | 攻撃・拘束・加速の罠を設置。`Z`で種類切替。 |

装備例と合成弾一覧は[攻略本](攻略本.md#4-装備構成例)を参照してください。

## 8. マップ

### 市街

道路、河川、林、ビルが混在する標準マップです。建物による射線管理、橋と通路の確保、壁破壊が重要です。

### 砂漠

古代要塞、崖、流砂、砂丘、オアシス、松明、ガス田があります。

- 昼は日陰・オアシス、夜は火の近くで環境トリオン消費を抑えられます。
- ガス田付近で爆発攻撃を使うと大爆発します。
- 雨は選択できません。

### 雪山神殿

寝殿造の廊下と大部屋が雪庭を囲むマップです。

- 雪庭ではトリオン消費が34%増加します。
- 障子は破壊後に再生します。
- 酒樽は爆発攻撃で誘爆します。
- 人魂はレーダー反応としてCPUを撹乱し、破壊後に再生します。
- 東西の神像は一定間隔で近くの隊員へトリオンを分けます。

### 地下通路

旅客ホーム、業務通路、線路、水路、汚泥区画を持つ地下マップです。

- 電光掲示板のカウントが0になると通過電車が走ります。
- 電車に接触すると即ベイルアウトします。
- ホームドアは両側連動です。
- ホームドア、水門、ブレーカーは近くで特殊操作キーを押して切り替えます。
- AIも敵側へ移動するためにホームドアを操作します。
- 水門を開くと水路の水位が段階的に下がります。

## 9. 防衛戦の設備と修復

### フラッグ修復

戦闘員はフラッグの近くで `F` を押し、自分のトリオンを注いで修復できます。オペレーターは支援パネルから遠隔修復を使用できます。

### 防衛設備

画面右下の防衛パネルから、防衛資材を消費して自動配置します。

| 設備 | 資材 | 役割 |
|---|---:|---|
| 防壁 | 22 | 3枚の壁で侵攻経路を遅らせる |
| 固定トラップ | 26 | 接近した侵攻兵へ範囲攻撃 |
| 固定砲台 | 40 | 周囲の侵攻兵を自動射撃 |
| 囮ビーコン | 16 | 一部の敵をフラッグから引き離す |

設備には耐久、使用上限、再使用時間、存続時間があります。敵にも破壊されます。CPUは安全な準備時間に不足設備を自動配置します。

## 10. オペレーター

チーム戦と防衛戦では、`O`から指示と支援を使用できます。

主な支援:

- 索敵
- 補給
- 照明
- 障壁
- 機動支援
- 囮
- フラッグ修復（防衛戦のみ）

位置指定型の支援は、支援ボタンを押したあと戦場をクリックまたはタップします。作戦画面はWASDまたは左パッドで移動できます。

## 11. AIの行動

全モードのAIは以下を共通で行います。

- 武器ごとの適正距離を維持
- 敵視、攻撃音、被弾、隠密状態から標的を選択
- 壁と遮蔽物を考慮した射線判定
- 複数経由点を使う簡易A*経路探索
- 地下通路のホームドア操作
- 長時間詰まった場合の破壊可能壁への攻撃
- 味方との重なり回避
- マップ固有の危険地帯と回復地点の利用

壁越しの攻撃は、通常弾、狙撃、近接、爆発、予告範囲、霧、炎床、フラッグ攻撃を含めて遮蔽されます。

## 12. ベイルアウト、撃破、復帰

- HPが0になるとベイルアウトします。
- `B`で自主ベイルアウトできます。
- 個人戦・チーム戦では一定時間後に復帰します。
- 防衛戦の侵攻ユニットは撃破後に復帰しません。
- 地下通路の通過電車へ接触した場合は即ベイルアウトします。

オンライン機能はSupabaseを設定した場合だけ有効になります。

主な機能:

- ユーザー名とパスワードによる登録・ログイン

## 18. よくある問題

### 画面が古いまま

ブラウザキャッシュが残っています。`Ctrl`＋`F5`、キャッシュ削除、またはプライベートウィンドウで確認してください。

### オンラインが「未設定」になる

`js/online-config.js`のSupabase URL、公開キー、`enabled`を確認してください。

### ユーザー登録に失敗する

Edge Functionとデータベーススキーマが未反映の可能性があります。[EDGE_FUNCTION_SETUP.md](EDGE_FUNCTION_SETUP.md)を確認してください。

### 操作キーが分からなくなった

試合中は `G` で操作ガイドを表示できます。出撃設定の「キー割り当て」から標準に戻せます。

### CPUが壁を攻撃している

迂回経路を複数回探索しても進めない場合に限り、破壊可能な壁を最終手段として攻撃します。ホームドアや破壊不能設備は攻撃対象になりません。
