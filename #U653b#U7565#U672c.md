## 1. 勝つための基本

### 射線が通る場所でだけ撃つ

壁は通常弾、狙撃、爆発、範囲攻撃を遮ります。敵の姿が見えても、攻撃起点と敵の間に壁があるなら攻撃を温存してください。

有効な動き:

- 壁の角から短時間だけ体を出す
- 敵が通路へ入る瞬間を狙う
- エスクードで敵の射線を切る
- メテオラで破壊可能な壁を崩して新しい射線を作る

### 適正距離を守る

射手、銃手、狙撃手は適正距離外で威力が下がります。

| 距離 | 向いている武器 |
|---|---|
| 至近～近距離 | スコーピオン、弧月、レイガスト、ショットガン、ハンドガン |
| 中距離 | 射手弾、アサルトライフル、ガトリング、グレネードガン |
| 遠距離 | イーグレット、ライトニング、アイビス |

スコープ中は照準色と距離表示を見て位置を調整します。

### トリオンを残す

攻撃へ使い切ると、シールド、緊急移動、フラッグ修復ができません。最大トリオンの20～30%は防御用に残すのが基本です。

### 同じ攻撃を連打しない

CPUは武器の距離へ合わせて移動します。次のような連携を使います。

- 直線弾で移動させてハウンド
- スパイダーで止めて狙撃
- 鉛弾で拘束して旋空・アイビス
- メテオラで退路を塞いで近接
- ダミービーコンで向きを変えて背後から攻撃

### 射点を変える

狙撃手や射手は同じ位置から撃ち続けないでください。1～2回攻撃したら別の遮蔽物へ移動します。

## 2. 能力配分

### バランス型

```text
トリオン 6 / 技術 6 / 戦闘 6
```

全武器を試しやすい標準配分です。

### 攻撃手型

```text
トリオン 4 / 技術 5 / 戦闘 9
```

移動、耐久、近接威力を重視します。グラスホッパーやシールド用のトリオンは残します。

### 射手型

```text
トリオン 8 / 技術 8 / 戦闘 2
```

弾数、射程、誘導、合成速度を重視します。接近対策としてシールドと移動トリガーを装備してください。

### 銃手型

```text
トリオン 6 / 技術 8 / 戦闘 4
```

命中、リロード、取り回しのバランスがよい構成です。

### 狙撃手型

```text
トリオン 6 / 技術 10 / 戦闘 2
```

遠距離精度を最大化します。発見されたら撃ち合わず、バッグワームやグラスホッパーで位置を変えます。

### 重装型

```text
トリオン 6 / 技術 3 / 戦闘 9
```

レイガスト、スラスター、ショットガンで前線を維持します。

### ビギナーズスキル

- **オートガード:** シールドかレイガストを必ず装備する。
- **エイム補正:** 銃手・狙撃手の入門向け。
- **倹約家:** 射手、工作手、防衛戦の長期戦向け。

## 3. 装備構成例

### 万能手

```text
MAIN: 弧月 / 旋空 / シールド / ARアステロイド
SUB : ARハウンド / バッグワーム / シールド / グラスホッパー
```

中距離はアサルトライフル、近距離は弧月、不利な時はグラスホッパーで離脱します。

### 二刀スコーピオン

```text
MAIN: スコーピオン / シールド / グラスホッパー / バッグワーム
SUB : スコーピオン / シールド / ダミービーコン / スパイダー
```

`C`のマンティスで射程を伸ばします。スパイダーやビーコンで向きを崩してから接近します。

### 合成射手

```text
MAIN: アステロイド / ハウンド / シールド / メテオラ
SUB : メテオラ / バイパー / シールド / バッグワーム
```

壁の裏で左右のキューブを準備し、`C`で合成します。

### 標準銃手

```text
MAIN: ARアステロイド / ARハウンド / シールド / スタアメーカー
SUB : HGアステロイド / バッグワーム / シールド / 鉛弾
```

中距離はアサルトライフル、接近されたらハンドガンへ切り替えます。

### 狙撃手

```text
MAIN: イーグレット / ライトニング / シールド / スパイダー
SUB : バッグワーム / ダミービーコン / シールド / グラスホッパー
```

動く敵はライトニング、遠い敵はイーグレット、大型敵や壁にはアイビスへの変更も有効です。

### 重装手

```text
MAIN: レイガスト / スラスター / シールド / SGアステロイド
SUB : メテオラ / エスクード / シールド / グラスホッパー
```

レイガスト盾で接近し、スラスターかショットガンで押し込みます。

### 工作手

```text
MAIN: スイッチボックス / ダミービーコン / シールド / スパイダー
SUB : バッグワームタグ / 空き / 空き / 空き
```

通路、扉、橋、フラッグ周辺へ罠を置きます。正面戦闘は避けてください。

## 4. トリガー別の要点

### 弧月・旋空

旋空は先端付近を当てると強力です。敵へ密着せず、一歩引いて最大射程を狙います。弧月は`Shift`＋発動でいなし攻撃を使えます。

### スコーピオン

`Shift`で普通・長・短を切り替えます。両手装備時は`C`でマンティスを使用できます。短時間の攻撃後に離脱する武器です。

### レイガスト・スラスター

レイガストは盾運用が中心です。相手の射撃を受け、リロードやキューブ準備へスラスターを合わせます。

### 射手キューブ

分割が少ないほど単発威力、分割が多いほど面制圧に向きます。

- 少分割: 単体、狭い通路、壁破壊
- 多分割: 高機動敵、複数敵、退路封鎖

### ハウンド

敵の現在位置ではなく、逃げ道の横へ照準を置くと回り込みやすくなります。

### バイパー

技術力が高いほど複雑な経路を取りやすくなりますが、壁を無視する弾ではありません。建物の角や通路入口へ経路を通します。

### 銃

- アサルトライフル: 中距離の継続火力
- ハンドガン: 近～中距離の取り回し
- ショットガン: 至近距離の面制圧
- グレネードガン: 集団と壁破壊
- ガトリング: 長時間の制圧

残弾が少ない時は、戦闘前に`Shift`＋発動で手動リロードします。

### 狙撃銃

- イーグレット: 万能型
- ライトニング: 高機動敵と削り
- アイビス: ボス、壁、静止目標

### 鉛弾

次に反対側から撃つ弾を、ダメージなし・シールド無視の拘束弾へ変えます。味方の追撃につなげる技です。

### スタアメーカー

隠密する敵へマーカーを付けます。バッグワーム、カメレオン、ボルボロス対策に有効です。

### スパイダー

強い設置場所:

- 狭い通路
- 建物の出口
- 地下ホームドア前
- フラッグへ向かう最短経路
- 雪山神殿の廊下

### エスクード

防御だけでなく、敵の分断や進路変更に使います。味方の通路も塞ぐため、地下通路では注意してください。

### ダミービーコン

敵と自分の間ではなく、敵の横や後ろへ置いて向きを変えます。

### スイッチボックス

`Z`で攻撃、拘束、加速を切り替えます。

- 攻撃: 敵が確実に通る場所
- 拘束: 狙撃やメテオラの補助
- 加速: 味方の突入と離脱

## 5. 合成弾

左右に対応する射手弾を準備し、`C`で合成します。

| 組み合わせ | 合成弾 | 用途 |
|---|---|---|
| アステロイド＋アステロイド | ギムレット | 高威力・貫通 |
| メテオラ＋バイパー | トマホーク | 曲射爆撃 |
| ハウンド＋メテオラ | サラマンダー | 追尾爆撃 |
| ハウンド＋ハウンド | ホーネット | 強追尾 |
| アステロイド＋バイパー | コブラ | 高速変化弾 |
| アステロイド＋メテオラ | 高速炸裂弾 | 高速小範囲爆発 |
| アステロイド＋ハウンド | 高速追尾弾 | 速度重視の追尾 |
| メテオラ＋メテオラ | 重炸裂弾 | 低速・大爆発 |
| バイパー＋バイパー | 多重変化弾 | 複数回の急旋回 |
| ハウンド＋バイパー | 変化誘導弾 | 設定軌道後に追尾 |

安全な合成手順:

1. 壁、エスクード、味方前衛の後ろへ入る。
2. MAIN側を展開する。
3. SUB側を展開する。
4. `C`で合成する。
5. 射線が通った瞬間だけ発射する。

## 6. モード別攻略

### 個人戦

瀕死の敵へ全員が集まりやすいため、混戦へ正面から入らず、削り合いが終わる瞬間を狙います。

優先順位:

1. 瀕死で孤立した敵
2. リロード中の銃手・狙撃手
3. 合成準備中の射手
4. 別の敵を追っている攻撃手
5. 防御中の高耐久敵

### チーム戦

基本編成:

- 前衛: 攻撃手または重装手
- 中衛: 万能手、銃手、射手
- 後衛: 狙撃手または工作手

オペレーター指示は、標的、退却、集合、支援の優先順位を揃えるために使います。

### 防衛戦

個人スコアよりフラッグを優先します。

ラウンド中:

- フラッグへ直行する敵を最優先
- 隊員を追う敵は前衛が引き受ける
- イルガーと遠距離ボスは後衛が処理
- 修復は敵をフラッグから離してから行う

ラウンド間:

- 手動リロード
- トリオン回復
- 設備補充
- 罠とワイヤーの再配置
- 狙撃位置の変更

### エキストラ

特殊形態は高耐久ですが、壁、射線、クールダウン、トリオン管理は隊員と共通です。

- 巨体は狭い通路で詰まりやすい
- 飛行型でも壁越し攻撃はできない
- いなし可能形態には小攻撃でいなしを使わせてから大技
- チーム戦では味方の射線を塞がない

## 7. マップ別攻略

### 市街

強い場所:

- 橋の出入口
- ビルの角
- 川沿いの長い射線
- 林の外周

狙撃手は川沿い、攻撃手は建物間、射手は交差点を使います。橋を渡る時はシールド、エスクード、ビーコンを先に出します。

### 砂漠

昼は日陰・オアシス、夜は松明付近を中継地点にします。安全地点へ居座らず、回復後は別の遮蔽物へ移動します。

ガス田では爆発攻撃が大爆発を起こします。敵、味方、フラッグとの距離を確認してからメテオラを使ってください。

### 雪山神殿

雪庭ではトリオン消費が増えるため、中央で長時間撃ち合わないでください。

- 障子を一時遮蔽にする
- 酒樽を連鎖爆発させる
- 人魂へCPUの注意を向ける
- 東西の神像でトリオンを補給
- 廊下へワイヤーと罠を設置

### 地下通路

- ホームドアを開ける前に電車のカウントを確認
- カウントが少ない時は線路へ入らない
- 水門操作後、水位が下がるまで待つ
- 長いホームでは狙撃とガトリング
- 業務通路ではショットガンと近接
- ホームドア前へワイヤーを置く

電車はHPに関係なく即ベイルアウトです。

## 8. 防衛戦の敵対策

### 設備の優先順位

序盤:

1. 防壁2～4組
2. 固定トラップ2基
3. 固定砲台1～2基
4. 囮ビーコン1基

設備をフラッグへ密集させると範囲攻撃で同時に壊されます。距離と方向を分散してください。

### モールモッド

近接主体です。ワイヤー、ショットガン、メテオラで集団処理します。

### イルガー

飛行、爆撃、自爆を使います。ライトニングやイーグレットで早期に処理し、自爆予告が出たらフラッグから離します。

### ラービット

高耐久、捕獲、キューブ化、突進を使います。鉛弾で止め、アイビスやギムレットを集中させます。

### 風刃

遠隔斬撃、伝播斬撃、多重斬撃、いなしを使います。

- 一列に並ばない
- 多重斬撃の予告後に横移動
- いなし中へ単発大技を撃たない

### 印

弾、錨、鎖、門と、盾、響、射、転送をBOOST付きで組み合わせます。

- 盾印へ正面射撃を続けない
- 錨・鎖を受けた味方を支援
- BOOSTが高い時は攻めず、消費させる

### アレクトール

生体弾、鳥型弾、針弾、キューブ波、捕獲領域を使います。

- 小弾を受け続けず遮蔽物へ
- 捕獲領域の中心から離れる
- いなしが切れた直後を狙う

### ボルボロス

6個のコアのうち本物は1個です。見た目では判別できません。

- 低コスト連射でデコイを消す
- 本物へアイビス、ギムレット、旋空
- 気体化中の毒範囲から離れる
- スタアメーカーで見失いを防ぐ

本物コアへの命中は4.4倍のボーナスダメージです。

### オルガノン

円軌道刃、多重円刃、外周刃、全周展開を使います。

- 軌道の内外を短く横切る
- 全周展開中は接近しない
- いなしを消費させてから大技

### 骸骨アタッカー

カメレオンと近接突進を使います。通路へワイヤーを置き、集団はメテオラで処理します。

### 骸骨シューター

キューブ展開中に狙撃します。集団へ近接突入しないでください。

### 骸骨スナイパー

バッグワーム、ビーコン、遮蔽物を使って接近します。鉛弾を受けた味方は早く援護します。

### 山狗

爪、連続引っ掻き、突進、咆哮、飛びかかりを使います。

- 正面に立たない
- 突進予告へ横回避
- 咆哮後の隙へ集中攻撃

### 夜鴉

黒い霧、羽ばたき、闇ブレス、羽根弾、滑空突進を使います。

- 霧の外へ出る
- ブレスを壁で切る
- 飛行中はライトニングとハウンド

### 白狐

瞬歩斬り、転移斬り、時計輪剣、鈍足剣を使います。

- 転移後に照準を反転
- 時計輪剣のリングから抜ける
- 鈍足中は回避よりシールド

### 猫又

双尾突き、妖尾狙撃、妖光レーザー、尾薙ぎ、跳躍を使います。

- レーザー予告を見たら射線外へ
- 跳躍着地地点から離れる
- 近距離では尾薙ぎの外へ回る

### 大蛇

長い20節の胴体を持ち、牙、落星火、極炎ブレス、巨体薙ぎ、炎纏い、終焉の炎を使います。

- 頭だけでなく胴体接触にも注意
- 落星火の着弾円から順番に離れる
- 極炎ブレスは壁で切る
- 終焉の炎では炎床の隙間を探す
- 覚醒後は尾側へ回る

## 9. エキストラ形態攻略

特殊形態も`1`～`4`でMAIN、`5`～`8`でSUBを選び、左右クリックで発動します。

### モールモッド

| MAIN | SUB |
|---|---|
| ブレード | 硬化 |
| 連続刃 | 跳躍 |
| 突進 | 後退斬り |
| 回転刃 | 捕食 |

小型と速度を生かし、壁や通路から接近します。

### イルガー

| MAIN | SUB |
|---|---|
| 爆撃 | 加速飛行 |
| 散開爆撃 | 背面迎撃 |
| 機銃 | 自爆予告 |
| 急降下 | 自爆 |

開けた場所で強い飛行型です。自爆は最後の手段です。

### ラービット

| MAIN | SUB |
|---|---|
| 打撃 | 装甲防御 |
| 捕獲腕 | 跳躍 |
| 連撃 | キューブ化 |
| 叩きつけ | 突進 |

高耐久で前線を維持し、拘束から味方の追撃へつなげます。

### 風刃

| MAIN | SUB |
|---|---|
| 遠隔斬撃 | いなし |
| 伝播斬撃 | 高速移動 |
| 近接斬撃 | 多重斬撃 |
| 斬撃陣 | 残弾集中 |

直線上の複数敵へ伝播斬撃、集団へ斬撃陣、単体へ残弾集中を使います。`C`でもいなしを発動できます。

### 印

| MAIN | SUB |
|---|---|
| 弾印 | 盾印 |
| 錨印 | 響印 |
| 鎖印 | 射印 |
| 門印 | 転送印 |

`C`でBOOST ×1、×2、×3を切り替えます。MAIN使用時に`Shift`を押すと、選択中のSUB印を掛け合わせます。

例:

- 弾印×盾印: 攻撃しながら防御
- 錨印×響印: 索敵付き拘束
- 鎖印×射印: 追加射出
- 門印×転送印: 転送付き展開

通常攻撃は×1、拘束と決め技は×2～×3が基本です。

### アレクトール

| MAIN | SUB |
|---|---|
| 生体弾 | いなし |
| 多重生体弾 | キューブ波 |
| 鳥型弾 | 装甲展開 |
| 針弾 | 捕獲領域 |

小弾で移動を制限し、捕獲領域へ追い込みます。

### ボルボロス

## 10. よくある敗因

### 攻撃が壁に吸われる

照準点だけでなく、キャラクターから命中地点までの実際の射線を確認してください。

### 射手キューブを完成できない

開けた場所で準備しています。壁、エスクード、味方の後ろで準備し、分割を欲張らないでください。

### リロード中に倒される

残弾0になる前に、移動中、ラウンド間、敵を見失った直後に手動リロードします。

### 近接で敵を追い続ける

敵そのものではなく退路へ向かいます。ワイヤー、ハウンド、味方射撃で逃げ道を限定してください。

### 防衛設備を密集させる

範囲攻撃で同時に壊されます。防壁、罠、砲台、囮を異なる距離と方向へ置きます。

### 大技をいなしされる

小攻撃を先に見せていなしを使わせ、効果終了後に狙撃、合成弾、ボス技を当てます。

### 地下鉄で即ベイルアウトする

掲示板を見ずに線路へ入っています。ホームドアが開いていても安全とは限りません。

### AIが壁を壊し始める

経路探索を繰り返しても進めない状態です。味方のエスクードや防壁が唯一の通路を塞いでいないか確認してください。
